# Discovery Call Script

## Goal

Qualify whether the client has enough technical material to benefit from an Overworker package.

## Questions

1. What are you building?
2. Where is the work stored?
3. What repos or demos exist?
4. What is working today?
5. What is broken or unclear?
6. Who needs to understand the work?
7. What decision should the package support?
8. What is the deadline?

## Qualification

Good fit:

- messy repo or archive
- real implementation attempt
- unclear positioning
- need for investor, customer, or partner clarity

Bad fit:

- pure idea with no artifacts
- request for guaranteed funding
- request for legal/tax/investment advice
- request to fabricate proof
