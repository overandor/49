# Security Policy

## Supported status

This repository is an early scaffold and should not be treated as production-secure yet.

## Sensitive data

Overworker may ingest repositories, chats, and project files that contain secrets or proprietary information. Production use requires:

- secret scanning
- redaction
- local-first processing option
- explicit user consent before uploading files to external services
- audit log of tool actions

## Reporting

Open a private issue or contact the maintainer before disclosing sensitive vulnerabilities publicly.
