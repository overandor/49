# Production Sale Checklist

A buyer or client should be able to answer these questions from the repo:

- What is it?
- Who buys it?
- What pain does it solve?
- What do they receive?
- What does it cost?
- What proof exists?
- What are the limits?
- What happens during delivery?
- What files are produced?
- What is not guaranteed?

## Required before sale

- Clear README
- Offer page
- Proposal template
- Sample report
- Before/after example
- Verification Firewall docs
- Overwork Score docs
- Terms, privacy, disclaimer
- Working CLI or demonstrable script
- At least one example output package
