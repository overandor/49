# Before / After Example

## Before

A technical founder has:

- five disconnected repos
- no clear README
- no demo script
- no risk disclosure
- no investor one-pager
- unsupported valuation language
- unclear product thesis

## After

Overworker produces:

- repo map
- product thesis
- README draft
- technical asset map
- investor one-pager
- risk disclosure
- Overwork Score
- 30/60/90 roadmap
- verification report

## Outcome

The work becomes easier to inspect, easier to demo, and easier to discuss with investors, customers, advisors, or collaborators.
