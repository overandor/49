# Contributing

Overworker is currently an early production-sale scaffold. Contributions should improve one of four areas:

1. Product clarity
2. Technical verification
3. Deliverable generation
4. Compliance/safety boundaries

## Rules

- Do not add unsupported investor, valuation, tax, legal, or trading-profit claims.
- Add tests for new source modules.
- Label claims as verified, inferred, user-provided, unsupported, or speculative.
- Preserve auditability of generated outputs.
