# Privacy Policy Draft

This is a draft privacy policy for product planning and must be reviewed before production use.

## Data processed

Overworker may process:

- repository metadata
- source files
- documentation
- chat archives
- project notes
- generated reports

## Sensitive information

Users should not upload secrets, credentials, regulated data, or private third-party information unless a production privacy/security agreement is in place.

## Retention

Production versions should define retention windows and deletion controls.

## External services

If LLM APIs or external tools are used, users must be informed before data is sent outside the local runtime.
